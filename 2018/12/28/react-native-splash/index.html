react-native-splash

react native 使用splash有很多方法，比如使用第三方package react-native-splash-screen,但是这种方法需要引入依赖，有的还需要link，一点都不native。所以现在记录一个android下使用原生splash的方法，iOS下很简单只要用xcode create splash 就行。

首先需要把作为splash的图片放到/res/drawable目录下。
在/values目录下的styles.xml中创建新的theme。

在配文件文件中使用修改mainactivity的theme。

然后就可以愉快的使用了。
